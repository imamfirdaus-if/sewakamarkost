## Customer Service
![Customer_Service](https://img.shields.io/badge/Customer%20Service-Kost%20Kita-green)

### Change directory to `customers`
```
cd customers
```

### Install modules
```
npm install
```

### Menjalankan Service
```
nodemon customers.js
```

### PORT
```
http://localhost:3000/
```
